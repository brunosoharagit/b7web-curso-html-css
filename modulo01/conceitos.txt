O que é HTML e CSS?
¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨
• HTML - estruturação da página web - código estruturacional
• CSS - estilização da página web - estilização
• Ambos trabalham juntos para aplicar a conversão do site

--> Navegador traduz o código HTML

--> botão "inspecionar": mostra as abas de funcionamento de uma página
    - pega e organiza, de forma legível, a estruturação do site, através da aba "Elements" (ou "Elementos")

O que são TAGs?
¨¨¨¨¨¨¨¨¨¨¨¨¨¨¨
• existem tags inseridas no HTML, semelhantes às caixas
• uma das tags mais utilizadas são headings (<h1></h1>)
    - variadas de 1 a 6
    - quanto maior o número, menor a tag

• tags utilizadas na demonstração
    - exemplos:
        a) <strong></strong> --> faz com que um elemento tenha fonte em negrito
        b) <hr/> --> linha horizontal: não há conteúdo
        c) <p>Qualquer texto não foi mal-digitado</p>
        d) <u></u> --> texto sublinhado
        e) ol e ul --> listas ordenada e não-ordenada, respectivamente

• a maior parte das tags possuem abertura e fechamento
    - com exceção da tag hr, br e tags que representam inputs
