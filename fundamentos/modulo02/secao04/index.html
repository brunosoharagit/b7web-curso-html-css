Cores e formas de inserção - CSS

Seção que mostra diferentes formas de inserir cores.
Como pode ser visto no exemplo ao lado, temos diferentes
maneiras de utilizar as cores na tag 'background-color'.

Maneiras de inserir cores

RGB

São utilizadas 3 cores primárias:

  • Red
  • Green
  • Blue

Hexadecimal

É utilizada números de 0 a 9,
e, posteriormente, letras de 'a' até 'f'.
